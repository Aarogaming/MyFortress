# Scripts Reference

All scripts and utilities for the AAS federation are housed in the **Workbench** repository for standardization and tooling.

- Primary scripts: [Workbench/scripts/](https://github.com/Aarogaming/Workbench/tree/main/scripts)
- Templates: [Library/templates/script_template.py](https://github.com/Aarogaming/Library/blob/main/templates/script_template.py)

For federation-wide utilities, use Workbench tools.
