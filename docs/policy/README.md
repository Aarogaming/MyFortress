# Policy Document Versioning

This directory contains policy documents for MyFortress. To ensure clarity and maintain an up-to-date record, the following versioning strategy is employed:

*   **Active Policies:** The most current version of a policy will have `_LATEST.md` appended to its filename (e.g., `POLICY_NAME_LATEST.md`). These are the active and authoritative policies.
*   **Archived Policies:** Older versions of policies, or policies that are no longer active, are moved to the `archive/` subdirectory. These are retained for historical reference only and should not be considered current.

When updating a policy, the old version (if dated) should be moved to the `archive/` directory, and the new version should be named with the `_LATEST.md` suffix.
