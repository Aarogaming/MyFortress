# Policy Pipeline Dry-Run

Use this copy-paste command sequence to run the full local policy pipeline and
generate all status artifacts in one pass.

```bash
SNAPSHOT_ID="local-$(date -u +%Y%m%dT%H%M%SZ)" && \
make cp4-status SNAPSHOT_ID="${SNAPSHOT_ID}" TREND_OUTPUT="artifacts/ci/policy_trend_${SNAPSHOT_ID}.json"
```

Expected outputs:

- `artifacts/ci/policy_pipeline_summary.json`
- `artifacts/ci/policy_gate_summary.txt`
- `artifacts/ci/policy_gate_negative_summary.txt`
- `artifacts/ci/cp3_readiness_summary.txt`
- `artifacts/ci/policy_trend_<snapshot-id>.json`
- `docs/policy/CHIMERA_V2_CP4_STATUS_AUTOGENERATED_2026-02-15.md`
